## 📖 Overview

DesignIQ AI is a modern web application designed to simulate an AI-powered website review experience. Users can enter a website URL to receive a comprehensive analysis dashboard featuring design scores, UI/UX insights, SEO evaluation, accessibility checks, performance metrics, and practical improvement suggestions.

The project focuses on delivering a clean, interactive, and responsive user experience while demonstrating modern frontend development practices.
